# LogCraft: Reprogramando o Mundo

Jogo digital educacional desenvolvido com **Phaser 3 + TypeScript** para trabalhar matemática e pensamento computacional nos anos iniciais.

## Conceito

O mundo digital foi corrompido pelo **Buggie**.  
O jogador deve restaurar os ecossistemas utilizando lógica, operações matemáticas e estratégias computacionais.

## Objetivos

- Desenvolver raciocínio lógico-matemático
- Trabalhar pensamento computacional
- Incentivar resolução de problemas
- Aprender através de debugging e experimentação

## Primeiro Mundo

### Cascata de Dados
Ecossistema focado em:
- restauração ambiental;
- fluxo de energia;
- lógica matemática;
- sistemas digitais.

## Tecnologias

- Phaser 3
- TypeScript
- Vite
- Zustand

## Executar Projeto

```bash
npm install
npm run dev
```
